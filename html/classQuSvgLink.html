◆ isValid()

bool QuSvgLink::isValid ( ) const

QuSvgLink::isValid returns true if the information stored is enough.

Validity
A link is valid if
  • its parent has an id
  • its parent is a valid node with a tag name
  • if an attribute is specified in the <link> element, the parent must have that attribute defined as well
  • if an attribute is not specified in the <link> element, then it must be clear from the context how to change the contents of the parent. For example a <text> element can host a <link> to change the text data.
Returns
True if the link contains valid information, false otherwise
