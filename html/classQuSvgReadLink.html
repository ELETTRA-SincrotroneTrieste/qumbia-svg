◆ isValid()

bool QuSvgReadLink::isValid ( ) const

QuSvgReadLink::isValid returns true if the information stored is enough.

Validity
A read element is valid if
  • its parent has an id
  • its parent is a valid node with a tag name
  • if an attribute is specified in the <read> element, the parent must have that attribute defined as well
  • if an attribute is not specified in the <read> element, then it must be clear from the context how to change the contents of the parent. For example a <text> element can host a <read> to change the text data.
Returns
True if the read element contains valid information, false otherwise
